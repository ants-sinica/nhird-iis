自動測試
********

本功能用來一次指定多個任務，自動依序執行。被選定的頁面只有第一個「變數事件」會被自動執行。

.. _h174fb648377959437b5c1f697c1c40:

使用方式
========

.. _h572187820253c7294643631303029:

設定並執行
----------

請利用右上角的選單開始自動測試執行視窗

\ |IMG1|\ 

執行視窗如下：

\ |IMG2|\ 

勾選左邊的方塊之後，按下方藍色執行按鈕，即可進行自動執行。

你可以保持本視窗在開啟狀態，執行結果會在本視窗直接顯示。也可以關閉本視窗，執行其他動作，需要了解執行狀態時再開啟。
